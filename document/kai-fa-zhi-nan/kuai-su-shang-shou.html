快速上手

快速上手

获取项目

通过公司git服务器获取最新的平台项目

git clone http://192.168.10.198:8800/r/zhx-vue-plateform.git

启动项目

我们一般习惯在第一次获取项目后首先执行以下代码来获取项目相关依赖

npm install

在获取完毕依赖后,我们即可启动项目了.目前我们只有admin可以开发.(你可以打开source/src/admin查看). 然后 我们即可启动项目了:

npm run dev:admin

启动完毕后,如果一切正常,你应该能看到控制台打印如下代码:

DONE  Compiled successfully in 23713ms

 I  Your application is running here: http://localhost:8090

OK! 大功告成, 我们可以通过以上地址访问项目了. :P

生产环境打包

我们通过以下指令即可完成生产环境的打包

npm run build:admin

打包文件结构分析

如果我们想查看打包后的文件结构,可以执行以下指令

npm run analyz:admin
